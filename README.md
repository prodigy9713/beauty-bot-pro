# BeautyBot Pro

Готовый шаблон Telegram-бота для бьюти-мастеров на Python, `aiogram 3` и `SQLite`.

## Что уже умеет бот

- показывает услуги красивыми карточками;
- показывает прайс и дает перейти к записи;
- записывает клиента на свободное время;
- скрывает занятые слоты;
- отправляет подтверждение клиенту;
- отправляет уведомление администратору;
- показывает отзывы, портфолио, адрес и контакты;
- дает администратору управлять услугами, окнами, отзывами и текстами бота.

## Быстрый запуск локально

1. Установите Python 3.11 или новее.
2. Создайте виртуальное окружение.
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Скопируйте `.env.example` в `.env`.
5. Укажите в `.env`:
   - `BOT_TOKEN`
   - `ADMIN_ID`
   - при необходимости `DATABASE_PATH`
6. Запустите бота:

```bash
python main.py
```

## Безопасность

- настоящий `.env` не должен попадать в GitHub;
- для репозитория используйте только `.env.example`;
- проверьте, что в `.gitignore` есть строка `.env`.
