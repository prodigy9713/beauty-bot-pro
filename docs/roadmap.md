# Roadmap

## Этап 1. База проекта

- структура папок и файлов;
- запуск бота;
- `/start`;
- клиентское меню;
- `/admin` с проверкой `ADMIN_ID`.

## Этап 2. Клиентский MVP

- каталог услуг;
- прайс;
- карточки услуг;
- отзывы, портфолио, адрес и контакты;
- запись клиента через выбор услуги, даты и времени.

## Этап 3. Админка

- просмотр записей;
- создание свободных окон;
- добавление, изменение и скрытие услуг;
- смена цен;
- управление отзывами;
- редактирование текстов шаблона.

## Этап 4. Готовность к продаже

- чистый README;
- инструкция по деплою;
- стартовые демо-данные;
- материалы для первых продаж;
- упаковка проекта как шаблона для мастеров.

## Следующие улучшения

- отмена и перенос записи;
- напоминания клиенту за день до визита;
- фото-портфолио прямо в боте;
- экспорт записей;
- несколько администраторов;
- переход на PostgreSQL при росте проекта.
